# x509-samples

Self-generated X.509 test fixtures for unit-testing the X509Parse and
X509Validity circom templates. These are NOT real QES certs — KICRF / EU MS
test certs land under `/fixtures/qes-certs/` once team-lead supplies them.

## leaf.der / leaf.pem

Self-signed RSA-2048 cert with both `notBefore` and `notAfter` encoded as
ASN.1 GeneralizedTime (forced by setting both dates past 2050 — RFC 5280
mandates GeneralizedTime for years ≥ 2050). Required because Phase 1
intentionally rejects UTCTime in `X509Validity.circom`.

To regenerate:

```bash
openssl req -x509 -newkey rsa:2048 \
  -keyout key.pem -out leaf.pem -nodes \
  -subj "/CN=QKB Test Leaf/O=Test QTSP/C=UA" -sha256 \
  -not_before "20510101000000Z" -not_after "20690101000000Z"
openssl x509 -in leaf.pem -outform DER -out leaf.der
rm key.pem        # private key is intentionally NOT committed
npx ts-node scripts/gen-x509-fixture.ts
```

Requires OpenSSL ≥ 3.2 for `-not_before` / `-not_after`.

## leaf.fixture.json

Generated by `scripts/gen-x509-fixture.ts`. Records the byte offsets of
the two GeneralizedTime fields inside `leaf.der` so the circom test can
feed exact `(bytes, offset)` pairs to `Asn1GeneralizedTime15`.
